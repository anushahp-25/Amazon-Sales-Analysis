# Amazon-Sales-Analysis (Interactive Dashboard creation using (Power BI)
## Project Objective
The Amazon Store wants to create an annual sales analysis report.
This report will help Amazon understand customer behavior, identify sales trends, and improve business performance for the coming years.

## Dataset used
- <a href = "https://github.com/anushahp-25/Amazon-Sales-Analysis/blob/main/Amazon_Combined_Data.xlsx">Dataset</a>

## Questions (KPIs)
- Compare the sales and orders using a single chart.
- Which month got the highest sales and orders?
- Who purchased more – Men or Women?
- What are the different order status types?
- List the top 10 states contributing to the sales.
- Relation between age group and number of orders.
- Which channel is contributing the most to the sales?
- Which is the highest-selling category?
- Percentage of total orders delivered.
